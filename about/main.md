# Riplexo — документация по устройству сайта

Riplexo — это веб-приложение для видеосвязи. Оно работает прямо в браузере без
установки приложений: пользователь создаёт комнату или входит в неё по коду и
начинает видеозвонок с демонстрацией экрана, управлением микрофоном/камерой и
индикацией говорящего.

Этот документ — оглавление. Подробное описание каждой части сайта вынесено в
отдельные файлы в этой же папке `about/`. Ниже — короткое описание того, что
находится в каждом файле.

> Документация описывает **только то, что уже реализовано в проекте**, без
> добавления новых функций.

## Краткая карта сайта

- **`/` — Главная (лендинг).** Маркетинговая страница с хедером, hero-секцией
  (две кнопки действия), блоком преимуществ и баннером о качестве 1080p.
  Подробно: [`landing-page.md`](./landing-page.md).
- **`/room/[roomId]` — Комната звонка.** Сам видеозвонок: панель с кодом
  комнаты и числом участников, область видео, панель управления
  (микрофон, камера, демонстрация экрана, выход).
  Подробно: [`room-page.md`](./room-page.md).

Переходы между этими страницами и сценарии входа описаны в
[`navigation-and-flows.md`](./navigation-and-flows.md).
